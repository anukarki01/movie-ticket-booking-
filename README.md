**Movie Ticket Booking System**


A console-based C application for managing movie ticket reservations with separate admin and user interfaces. Features secure login, seat booking/cancellation, ticket generation, and file-based data storage.


**Features**
 **Admin System**

 
Secure Login: Admin authentication with username/password

View Reserved Seats: See all booked seats with customer details

Price Management: Change ticket prices dynamically

Security: Limited login attempts with warnings



 **User System**

 
Secure Login: User authentication system

Movie Selection: Choose from 3 available movies

Seat Reservation: Interactive seat selection with visual availability

Ticket Cancellation: Cancel existing reservations

Digital Tickets: Generate formatted booking tickets





**Security Features**


Maximum 3 login attempts

Warning after 2 failed attempts

Input validation for all user entries

Buffer overflow protection

**Limitations**


Data persists only during program execution (except file saves)

No database integration

Basic console interface

Fixed movie selection
